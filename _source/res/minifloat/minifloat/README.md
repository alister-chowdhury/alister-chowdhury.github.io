# MiniFloat

Helpers for converting to and from minifloat formats.
Implementation is C++, but interface is C.

## 4bit formats
* e2m1
* binary4p2sf
* binary4p2se

## 8bit formats
* fp8 (1.5.2)
* e4m3 (OCP-E4M3, 1.5.2)
* e5m2 (OCP-E5M2, 1.5.2)

## 16bit formats
* bfloat16 (1.8.7)
* f16 (half, 1.5.10)

## Other
* u11 (R11G11B10, 0.5.6)
* u10 (R11G11B10, 0.5.5)
